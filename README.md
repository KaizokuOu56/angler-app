# ***angl.er*** : Github repository of team Chaar-mander for Hackenza 26'
Project aims to build an app that allows 2 way communication between devices using the flashlight and camera only.

# Planned goals
- Prototype (0.1.0) : build a proof of concept
- Non-encrypted version
- String compression and dictionary
- Encryption
- Working app
- Stage 2 : handle movement
