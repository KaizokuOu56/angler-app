# v0.1.0
- Prototype / proof of concept
- Homescreen with option to choose sender / receiver
- Sender screen has area to enter text, then changes to morse and strobes it
- Receiver screen has camera always on, and processes the strobed morse back into the message. Can choose to start / stop parsing and clear the message area
